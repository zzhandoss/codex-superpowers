# codex-superpowers

`v2` baseline skills for the Codex-first workflow.

## Structure

- `skills/using-superpowers`: startup routing and enforcement
- `skills/brainstorming`: product/design framing
- `skills/engineering-design`: engineering contract for planning
- `skills/ui-ux-design`: UI/UX design and revision flow
- `skills/placeholder-*`: deferred decision lifecycle
- `skills/writing-plans`: initiative, workstream, and execution planning
- `skills/execution-*`: inline and delegated execution
- `skills/systematic-debugging`: evidence-first debugging
- `skills/verification-before-completion`: completion evidence gate
- `skills/requesting-code-review`: review request preparation
- `skills/receiving-code-review`: review intake and resolution
- `skills/finish-work`: final closeout

## Notes

- This repository currently contains only the `v2` skill set.
- Planning templates live under `skills/writing-plans/`.
